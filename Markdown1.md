# 00 - Introdução ao Python

## 1. O que é Python

Python é uma linguagem de programação de alto nível, interpretada, de propósito geral e multiplataforma. Ela foi projetada com foco em simplicidade, legibilidade e produtividade, permitindo que desenvolvedores escrevam código claro e conciso.

Atualmente, Python é amplamente utilizada em diferentes áreas da computação, como:

- Desenvolvimento web
- Automação de tarefas
- Análise e ciência de dados
- Inteligência artificial e Machine Learning
- Computação científica
- Educação e ensino de programação

Sua sintaxe simples torna a linguagem especialmente adequada para iniciantes, ao mesmo tempo em que oferece recursos poderosos para aplicações profissionais e industriais.

---

## 2. Histórico do Python

A linguagem Python foi criada no final da década de 1980 por **Guido van Rossum**, no *Centrum Wiskunde & Informatica (CWI)*, na Holanda.

O nome **Python** vem do grupo humorístico britânico *Monty Python*.

- Primeira versão: **Python 0.9.0 (1991)**
- Versão moderna: **Python 3.0 (2008)**
